## 📖 Project Overview

StudyFlow AI is an **imaginary AI-powered study assistant** designed for learning purposes only.

The goal of this project was to design and build a landing page as if the product were real, focusing on:

- Presenting a product idea
- Structuring a modern landing page
- Practicing UI layout and design implementation

---

## ✨ Features of the Website

- Responsive landing page layout  
- Hero section with product introduction  
- Features section with cards  
- Benefits ("Why choose us") section  
- Pricing section  
- Embedded video section  
- Footer with navigation and social links  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- CSS Grid
